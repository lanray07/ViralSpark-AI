# Privacy Policy

ViralSpark AI stores onboarding choices, generated content, saved library items, content plans, and planned posts locally on the user's device with SwiftData.

When live AI is enabled, generation requests are sent to the configured secure backend endpoint. The iOS app must not contain OpenAI API keys or other provider secrets.

Users should not enter sensitive personal, financial, legal, medical, or confidential information into prompts. Local data can be deleted from Settings.
